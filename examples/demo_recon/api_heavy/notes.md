# API Heavy Notes

- API-style endpoints include object-like identifiers such as `user_id`, `account_id`, `org_id`, and `tenant_id`.
- Repeated export URLs should be deduplicated later.
- Docs and GraphQL-looking hosts are interesting context but should not be treated as confirmed issues.
- `internal-docs.example-bounty.test` returned forbidden in the captured data, so any later report should avoid claiming access.
- Dev and staging hostnames may deserve manual review when authorised, but the dataset does not show a confirmed weakness.
